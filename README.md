This is a simple NLP web app that performs sentiment analysis on movie reviews using a Naive Bayes classifier trained on TF-IDF features.
Features
- Pre-trained sentiment model (positive, neutral, negative)
- Web interface built with Flask and HTML/CSS
- Local deployment and easy extension for real-world data

 
How to Run Locally
bash
pip install -r requirements.txt
python train_model.py
python app.py

Then visit: http://127.0.0.1:5000/
